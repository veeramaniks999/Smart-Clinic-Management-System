# User Stories

## Doctor

### View Appointments
As a Doctor, I want to view my daily appointments so that I can manage my schedule efficiently.

**Acceptance Criteria:**
- [ ] Doctor can see today's appointments
- [ ] Appointment details include patient name, time, and reason
- [ ] Past appointments are archived

**Priority:** High  
**Story Points:** 3

---

### Update Patient Records
As a Doctor, I want to update patient medical records so that accurate information is maintained.

**Acceptance Criteria:**
- [ ] Doctor can add diagnosis, prescription, and notes
- [ ] Records are saved and retrievable
- [ ] Only authorized doctor can update records

**Priority:** High  
**Story Points:** 3

---

### Send Prescriptions
As a Doctor, I want to send prescriptions to patients digitally so that they can receive them conveniently.

**Acceptance Criteria:**
- [ ] Prescriptions can be sent via email or app
- [ ] Patient receives confirmation
- [ ] Only authorized doctor can send

**Priority:** Medium  
**Story Points:** 2

---

## Patient

### Book Appointment
As a Patient, I want to book an appointment online so that I don’t have to wait in the clinic.

**Acceptance Criteria:**
- [ ] Patient selects doctor and time slot
- [ ] Confirmation is sent via email/notification
- [ ] Appointment appears in patient dashboard

**Priority:** High  
**Story Points:** 3

---

### View Medical History
As a Patient, I want to view my medical history so that I can track my health progress.

**Acceptance Criteria:**
- [ ] Patient can see past visits, prescriptions, and diagnoses
- [ ] Data is secure and private
- [ ] Only authorized patient can view their own data

**Priority:** Medium  
**Story Points:** 2

---

### Receive Appointment Notifications
As a Patient, I want to receive notifications for upcoming appointments so that I don’t miss them.

**Acceptance Criteria:**
- [ ] Notifications sent via email or app
- [ ] Reminders sent 24 hours and 1 hour before appointment
- [ ] Only relevant appointments trigger notifications

**Priority:** Medium  
**Story Points:** 2

---

## Admin

### Manage Users
As an Admin, I want to manage doctor and patient accounts so that the system remains organized.

**Acceptance Criteria:**
- [ ] Admin can add, remove, and edit users
- [ ] Roles are assigned correctly
- [ ] User activity logs are maintained

**Priority:** High  
**Story Points:** 3

---

### View Appointment Statistics
As an Admin, I want to view appointment statistics so that I can analyze clinic operations.

**Acceptance Criteria:**
- [ ] Stats include total appointments, cancellations, and doctor-wise breakdown
- [ ] Data is displayed in charts/tables
- [ ] Only authorized admin can view

**Priority:** Medium  
**Story Points:** 2

---

### Handle Cancellations and Rescheduling
As an Admin, I want to handle cancellations and rescheduling so that the workflow remains smooth.

**Acceptance Criteria:**
- [ ] Admin can cancel or reschedule appointments
- [ ] Patients and doctors are notified
- [ ] Conflicts are checked automatically

**Priority:** Medium  
**Story Points:** 2
